# I2CStepper I2C Protocol v2

Документ описывает текущий регистровый интерфейс I2CStepper для интеграции с Samovar.

## Общие правила

- Размер массива регистров: `48` байт.
- Версия протокола: `2`.
- Многобайтные значения передаются старшим байтом вперед: регистр с суффиксом `_H` хранит старший байт.
- `role` - I2C-роль/адрес устройства: `1` для `I2CMIXER`, `2` для `I2CPUMP`.
- `mode` - рабочий режим: `1` `Mixer`, `2` `Pump`, `3` `Fill`.
- `Fill` работает только при роли `I2CPUMP`; роль `I2CFILLING` не используется как I2C-адрес.
- После записи параметров управляющая сторона должна отправить команду через `REG_COMMAND` и увеличить `REG_COMMAND_SEQ`. I2CStepper выполнит команду, запишет тот же номер в `REG_ACK_SEQ` и очистит `REG_COMMAND`.

## Режимы

| Значение | Константа | Назначение | Единицы скорости | Единицы цели |
| --- | --- | --- | --- | --- |
| `1` | `I2CMIXER` | Мешалка | об/мин | секунды |
| `2` | `I2CPUMP` | Насос без заданного объема | мл/ч | нет, остановка командой |
| `3` | `I2CFILLING` | Дозированный налив | мл/ч | мл |

`Mixer` поддерживается только при роли `I2CMIXER`. `Pump` и `Fill` поддерживаются только при роли `I2CPUMP`.

## Регистры

| Регистр | Индекс | Доступ | Описание |
| --- | ---: | --- | --- |
| `REG_MAGIC` | `0` | R | Маркер EEPROM/protocol: `0x53`. |
| `REG_VERSION` | `1` | R | Версия протокола: `2`. |
| `REG_CAPS` | `2` | R | Битовая маска возможностей устройства. |
| `REG_ROLE` | `3` | R | Текущая I2C-роль/адрес: `I2CMIXER` или `I2CPUMP`. |
| `REG_MODE` | `4` | R/W | Рабочий режим: `Mixer`, `Pump` или `Fill`. |
| `REG_COMMAND` | `5` | W/R | Команда к исполнению; после обработки сбрасывается в `0`. |
| `REG_COMMAND_SEQ` | `6` | W | Номер команды. Меняется управляющей стороной после записи `REG_COMMAND`. |
| `REG_ACK_SEQ` | `7` | R | Номер последней обработанной команды. |
| `REG_STATUS` | `8` | R | Битовая маска текущего состояния. |
| `REG_ERROR` | `9` | R | Код последней ошибки. |
| `REG_RELAY_MASK` | `10` | R/W | Младшие 4 бита управляют четырьмя реле. |
| `REG_SENSOR_FLAGS` | `11` | R/W | Флаги внешнего датчика. |
| `REG_OPTION_FLAGS` | `12` | R/W | Флаги поведения шаговика. |
| `REG_MIXER_RPM_H/L` | `13..14` | R/W | Скорость `Mixer`, об/мин. |
| `REG_MIXER_RUN_H/L` | `15..16` | R/W | Время работы `Mixer`, секунды. `0` означает работу до команды `STOP`. |
| `REG_MIXER_PAUSE_H/L` | `17..18` | R/W | Пауза `Mixer` после завершения рабочего интервала, секунды. |
| `REG_PUMP_MLH_H/L` | `19..20` | R/W | Скорость `Pump`, мл/ч. |
| `REG_PUMP_PAUSE_H/L` | `21..22` | R/W | Пауза `Pump` по внешнему датчику, секунды. |
| `REG_FILL_ML_H/L` | `23..24` | R/W | Целевой объем `Fill`, мл. |
| `REG_FILL_MLH_H/L` | `25..26` | R/W | Скорость `Fill`, мл/ч. |
| `REG_STEPS_PER_ML_H/L` | `27..28` | R/W | Калибровка насоса, шагов на 1 мл. |
| `REG_REMAINING_3..0` | `29..32` | R | Остаток: секунды для `Mixer`, мл для `Pump`/`Fill`. |
| `REG_CURRENT_SPEED_H/L` | `33..34` | R | Текущая скорость в шагах/с; `0`, если шаговик не выполняет движение или стоит на паузе. |
| `REG_LOCK` | `47` | internal | Служебный регистр блокировки обмена. |

Регистры `35..46` зарезервированы.

## Возможности

`REG_CAPS`:

| Бит | Константа | Значение |
| ---: | --- | --- |
| `0x01` | `I2CSTEPPER_CAP_MIXER` | Поддерживается `Mixer`. |
| `0x02` | `I2CSTEPPER_CAP_PUMP` | Поддерживается `Pump`. |
| `0x04` | `I2CSTEPPER_CAP_FILLING` | Поддерживается `Fill`. |
| `0x08` | `I2CSTEPPER_CAP_RELAY` | Доступно управление реле. |
| `0x10` | `I2CSTEPPER_CAP_SENSOR` | Доступен внешний датчик. |

## Статус и ошибки

`REG_STATUS`:

| Бит | Константа | Значение |
| ---: | --- | --- |
| `0x01` | `I2CSTEPPER_STATUS_RUNNING` | Шаговик выполняет движение. |
| `0x02` | `I2CSTEPPER_STATUS_PAUSED` | Активна пауза. |
| `0x04` | `I2CSTEPPER_STATUS_SENSOR` | Внешний датчик активен. |
| `0x08` | `I2CSTEPPER_STATUS_CALIBRATION` | Идет калибровка насоса. |
| `0x80` | `I2CSTEPPER_STATUS_ERROR` | В `REG_ERROR` есть ошибка. |

`REG_ERROR`:

| Код | Константа | Значение |
| ---: | --- | --- |
| `0` | `I2CSTEP_ERR_NONE` | Ошибки нет. |
| `1` | `I2CSTEP_ERR_UNSUPPORTED_MODE` | Режим не поддерживается текущей ролью устройства. |
| `2` | `I2CSTEP_ERR_BAD_CONFIG` | Некорректная конфигурация движения: нулевая скорость или нулевая цель. |
| `3` | `I2CSTEP_ERR_COMM_TIMEOUT` | Сработал fail-safe по зависанию I2C. |

## Флаги

`REG_OPTION_FLAGS`:

| Бит | Константа | Значение |
| ---: | --- | --- |
| `0x01` | `I2CSTEPPER_FLAG_REVERSE_AFTER_PAUSE` | В `Mixer` менять направление после паузы. |
| `0x02` | `I2CSTEPPER_FLAG_SMOOTH_START` | Использовать плавный старт. |
| `0x04` | `I2CSTEPPER_FLAG_DIRECTION` | Направление движения. |

`REG_SENSOR_FLAGS`:

| Бит | Константа | Значение |
| ---: | --- | --- |
| `0x01` | `I2CSTEPPER_SENSOR_ACTIVE_HIGH` | Датчик активен при HIGH. Без флага активным считается LOW. |
| `0x02` | `I2CSTEPPER_SENSOR_STOP` | Активный датчик останавливает шаговик. |
| `0x04` | `I2CSTEPPER_SENSOR_PUMP_PAUSE` | В `Pump` активный датчик ставит насос на паузу на `REG_PUMP_PAUSE`. |

Если одновременно включен `I2CSTEPPER_SENSOR_STOP`, он имеет приоритет над паузой.

## Команды

| Код | Команда | Действие |
| ---: | --- | --- |
| `0` | `I2CSTEP_CMD_NONE` | Нет команды. |
| `1` | `I2CSTEP_CMD_APPLY` | Применить записанные регистры к текущей сессии без сохранения в EEPROM. |
| `2` | `I2CSTEP_CMD_START` | Применить регистры и запустить текущий режим. |
| `3` | `I2CSTEP_CMD_STOP` | Остановить шаговик. |
| `4` | `I2CSTEP_CMD_SAVE` | Применить регистры и сохранить конфигурацию в EEPROM. |
| `5` | `I2CSTEP_CMD_CALIBRATE_START` | Запустить калибровку насоса. Доступно только при роли `I2CPUMP`. |
| `6` | `I2CSTEP_CMD_CALIBRATE_FINISH` | Завершить калибровку, остановить насос и пересчитать `STP/ML` по 100 мл. |
| `7` | `I2CSTEP_CMD_RELAY` | Применить маску реле и связанные регистры сессии. |

Типовой порядок обмена:

1. Записать нужные параметры и `REG_COMMAND`.
2. Увеличить `REG_COMMAND_SEQ`.
3. Дождаться, пока `REG_ACK_SEQ` станет равен новому `REG_COMMAND_SEQ`.
4. Проверить `REG_STATUS` и `REG_ERROR`.

## EEPROM v2

Структура EEPROM v2 хранит:

- маркер и версию;
- `role` и `mode`;
- параметры `Mixer`, `Pump`, `Fill`;
- `STP/ML`;
- флаги опций и внешнего датчика;
- маску реле.

При несовпадении маркера или версии EEPROM инициализируется значениями по умолчанию: `Mixer`, 20 об/мин, `Pump` 100 мл/ч, `Fill` 100 мл при 100 мл/ч, `STP/ML = 16000`, плавный старт включен, внешний датчик настроен на остановку.
